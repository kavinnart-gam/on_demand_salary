## Step 1: How to use

1.Download or clone this repo.

2.Install dependencies.


```bash
# using npm
npm install

# OR using Yarn
yarn install
```

2.Run project
   - For iOS

```bash
# using npm
npm run ios 

# OR using Yarn
yarn ios
```

   - For Android

```bash
# using npm
npm run android 

# OR using Yarn
yarn android
```
